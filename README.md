# deployer-demo

A demo SvelteKit app showcasing the stack:

- **SvelteKit** (Svelte 5 with runes)
- **Tailwind CSS v4**
- **shadcn-svelte** components (vega preset)
- **TypeScript**
- **Bun** as package manager
- **mode-watcher** for system-aware dark mode

The single landing page demonstrates a nav with theme toggle, a hero with CTAs, and a six-card features grid.

## Develop

```sh
bun install
bun run dev
```

Open http://localhost:5173.

## Build

```sh
bun run build
bun run preview
```

## Type-check

```sh
bun run check
```
